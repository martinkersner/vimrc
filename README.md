# vimrc
vimrc inspired by [Amir Salihefendic](https://github.com/amix/vimrc)

## Additional scripts
* [tinymode.vim](http://www.vim.org/scripts/script.php?script_id=2223)

## How to install
```
cd ~
git clone https://github.com/martinkersner/vimrc.git
mv vimrc/ .vim/
ln -s .vim/.vimrc .vimrc
cd .vim/
chmod +x run.bash
./run.bash
```
